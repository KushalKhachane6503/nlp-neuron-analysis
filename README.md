#  Neuron Importance Analysis in Neural Networks

##  Overview
This project analyzes how neuron importance behaves under different training conditions in neural networks.

##  Objective
- Understand neuron importance
- Compare importance rankings across different training setups
- Analyze correlation and overlap between rankings

##  Methodology
- Train neural network models
- Compute neuron importance scores
- Compare rankings under different conditions
- Perform analysis on correlation and overlap

##  Results
- Observed very low correlation between neuron rankings
- Found minimal overlap across different training conditions
- Indicates instability in neuron importance

##  Tech Stack
- Python
- TensorFlow / PyTorch
- NumPy
- Pandas
- Matplotlib

##  How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the notebook:
   jupyter notebook

##  Project Structure
- notebooks/ → experiments
- results/ → outputs and plots
